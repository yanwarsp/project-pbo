# project-pbo
Project UAS Pemrograman Berorientasi Objek 

## Aplikasi Simulasi Penilaian Mata Kuliah dan Menghitung Indeks Prestasi

> Requirements
- JDK 8
- SceneBuilder
- Laragon
- JavaFX SDK

> Gambaran Aplikasi
Tampilan awal terdapat 4 buah menu yaitu :
- Buku Nilai
- Indeks Presasi
- Bantuan
- Tentang Kami
