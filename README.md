# project-pbo
Project UAS Pemrograman Berorientasi Objek 

## Aplikasi Simulasi Penilaian Mata Kuliah dan Menghitung Indeks Prestasi

> Requirements
- JDK 8
- SceneBuilder
- Laragon
- JavaFX SDK

> nama database project-pbo
tabel 1 : nilaimhs
tabel 2 : indeksprestasi

> Gambaran Aplikasi
Tampilan awal terdapat 4 buah menu yaitu :
- Buku Nilai
- Indeks Prestasi
- Bantuan
- Tentang Kami

(p = persentase)
